Kawaii.js

A visual novel engine.

Basics

Creating a novel

To create a novel you will need to create a html file. Let's call it 'index.html'. Then connect Kawaii.js. Then you should write novel script and start the novel using .start() method.

Contents of index.html:
<DOCTYPE html> <html> <head> <title>Example Novel</title> </head> <body> <div class=Kawaii ></div> <script src="libs/Kawaii.js" ></script> <script id=Script type="application/x-vnd.kawaii.acts"></script> <script></script> </html>
